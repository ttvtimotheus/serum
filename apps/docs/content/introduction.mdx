# Serum UI

Serum UI is a modern, accessible React component library built on top of Tailwind CSS. It provides a set of beautifully crafted components that you can use to build fast, responsive web applications.

## Philosophy

Our goal is to provide components that are:

- **Accessible by default** - All components follow WAI-ARIA guidelines
- **Customizable** - Easy to theme and extend with Tailwind CSS
- **Developer friendly** - Simple APIs with TypeScript support
- **Modern** - Built with the latest React patterns and best practices

## Getting Started

To get started with Serum UI, check out our [Installation](/installation) guide or jump straight into exploring our [Components](/components/button).
